# Assets guide

Place your images from Instagram Highlights here. Suggested folders:

- assets/menu/ — Main menu dishes (from Highlight "Menyu")
- assets/breakfast/ — Breakfast items (from Highlight "Səhər yeməyi")
- assets/shisha/ — Shisha menu images (from Highlight "Shisha Menu")
- assets/gallery/ — General gallery images for the Gallery section

Recommended file names (used in script.js by default, adjust as needed):

- assets/menu/plov.jpg
- assets/menu/dushbara.jpg
- assets/menu/kyufta.jpg
- assets/menu/dolma.jpg
- assets/menu/coban.jpg
- assets/menu/baki.jpg
- assets/menu/paxlava.jpg
- assets/menu/sekerbura.jpg
- assets/menu/ayran.jpg
- assets/menu/tea.jpg

- assets/breakfast/set.jpg
- assets/breakfast/kuku.jpg

- assets/shisha/classic.jpg
- assets/shisha/mint.jpg
- assets/shisha/berry.jpg

- assets/gallery/01.jpg .. 08.jpg (optional)

If your file names differ, update paths in `script.js` (menuData) and in `index.html` for the Gallery images.
